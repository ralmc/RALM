Portafolio de proyectos:

· Plataforma que conecta a estudiantes con empresas para realizar proyectos reales.

· Página web para venta de piezas de autos chocados, confiable y accesible.

· Plataforma web y móvil para automatizar tareas de microempresas y PYMES.

· Gestión de proyectos académicos para seguimiento de tareas y avances de estudiantes.

· Tablero de seguimiento que muestra la participación de estudiantes en cada proyecto.
